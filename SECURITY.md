# Security Policy

## Supported versions

Until the first public release, only the current default branch is supported.

## Reporting a vulnerability

Do not post credentials, private facility information, or exploit details in a public issue. Use the repository owner's private security-reporting channel when it becomes available. Until then, provide only a minimal public issue asking for a private contact route.

This application must not connect to SCADA, plant controls, intranets, or operational systems. A report that discovers such a connection is release-blocking.
